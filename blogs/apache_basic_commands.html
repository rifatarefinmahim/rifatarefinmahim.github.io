Basic Commands of Apache Server

01-Jan-2045

Apache HTTP Server (or simply Apache) is an open-source web server software developed by the Apache Software Foundation. It's widely used to host and serve websites and web applications on Linux systems.

Key Features

  • Open-source & Free
  • Modular Architecture (e.g., mod_ssl, mod_rewrite)
  • Cross-platform
  • .htaccess Support
  • Virtual Hosts support
  • Strong Logging and Security features

Install Apache
sudo apt update
sudo apt install apache2
Start/Stop/Restart Apache
sudo systemctl status apache2
sudo systemctl start apache2
sudo systemctl stop apache2
sudo systemctl restart apache2
Enable at Boot
sudo systemctl enable apache2

Default Web Directory

Default web directory of apache webserver is /var/www/html. Place your index.html or PHP files here.

Configuration Files

  • Main Config - /etc/apache2/apache2.conf
  • Virtual Hosts - /etc/apache2/sites-available/000-default.conf
  • Ports Config - /etc/apache2/ports.conf
  • Enabled Modules - /etc/apache2/mods-enabled/

Enable modules with

sudo a2enmod rewrite
sudo systemctl restart apache2

Test Apache

After installing Apache, visit -

  • http://localhost
  • http://your_server_ip

You should see the "Apache2 Ubuntu Default Page".

Security Tips

  • Disable unnecessary modules
  • Use HTTPS (e.g., via Let's Encrypt)
  • Disable directory listing
  • Set proper file permissions and use .htaccess rules
